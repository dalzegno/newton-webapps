Bish-Bosh!

Bish-bosh lists numbers between 0 and what you choose in the Box input slider.

If the number is dividable by the Bish number 'Bish' is shown instead of the number.

If the number is dividable by the Bosh number 'Bosh' is shown instead of the number.

If the number is dividable by both numbers 'Bish-Bosh' is shown instead of the number.

Box input

Bish

Bosh
